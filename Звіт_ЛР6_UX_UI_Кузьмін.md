| **Serif** | Антиква (зарубковий шрифт) | Шрифти з декоративними зарубками на кінцях літер |

### 2. Відповіді на питання

#### 2.1 Дослідити основні типи шрифтів Serif / Sans-serif / Monospace / Display

**Serif (Антиква)**
- **Приклади шрифтів:** Times New Roman, Playfair Display, Georgia, Merriweather
- **Особливості:** Мають декоративні зарубки на кінцях літер, створюють відчуття традиційності та надійності
- **Використання:** Друковані тексти, книги, газети, преміальні бренди, академічні сайти

**Sans-serif (Гротеск)**
- **Приклади шрифтів:** Inter, Roboto, Open Sans, Helvetica, Montserrat
- **Особливості:** Шрифти без зарубок, сучасний вигляд, відмінна читабельність на екранах
- **Використання:** Вебсайти, мобільні застосунки, інтерфейси користувача, UI-дизайн

**Monospace (Моноширинні)**
- **Приклади шрифтів:** JetBrains Mono, Courier New, Fira Code, Consolas
- **Особливості:** Усі символи мають однакову ширину, асоціюються з кодом та технологіями
- **Використання:** Програмування, відображення коду, технічна документація, термінали

**Display / Decorative (Акцидентні)**
- **Приклади шрифтів:** Bebas Neue, Lobster, Pacifico, Impact
- **Особливості:** Декоративні шрифти з унікальним характером, призначені для заголовків
- **Використання:** Логотипи, заголовки, брендинг, рекламні матеріали (НЕ для основного тексту)

#### 2.2 Дослідити основні типи кольорових схем

**Monochromatic (Монохромна)**
- **Принцип формування:** Використання відтінків одного кольору (зміни насиченості та яскравості)
- **Приклад палітри:** Синій → Світло-синій → Темно-синій
- **Враження:** Стримано, професійно, мінімалістично
- **Застосування:** Корпоративні сайти, бізнес-застосунки

**Analogous (Аналогова)**
- **Принцип формування:** Кольори, що розташовані поруч у кольоровому колі (наприклад, синій + блакитний + бірюзовий)
- **Приклад палітри:** Синій → Блакитний → Бірюзовий
- **Враження:** Спокій, гармонія, природність
- **Застосування:** Природні теми, wellness-застосунки, релаксаційні продукти

**Complementary (Комплементарна)**
- **Принцип формування:** Контрастні кольори, розташовані один навпроти одного в колірному колі (наприклад, синій + помаранчевий)
- **Приклад палітри:** Синій ↔ Помаранчевий
- **Враження:** Енергія, динаміка, акцент
- **Застосування:** Кнопки CTA, акції, знижки, яскраві бренди

**Triadic (Тріадна)**
- **Принцип формування:** Три кольори, рівномірно розташовані на колірному колі (наприклад, червоний + жовтий + синій)
- **Приклад палітри:** Червоний ↔ Жовтий ↔ Синій
- **Враження:** Яскраво, збалансовано, креативно
- **Застосування:** Дитячі продукти, креативні бренди, розважальні застосунки

#### 2.3 Типографічна ієрархія

**Типографічна ієрархія (Typographic Hierarchy)** — це система організації текстових елементів за рівнем важливості та функціональністю.

**Коли застосовувати:**
- При проєктуванні будь-якого інтерфейсу з текстовим контентом
- Для створення чіткої структури інформації
- Для покращення читабельності та сканування контенту

**Основні елементи ієрархії:**
1. **H1 (Заголовок першого рівня)** — 24-32px, Bold — назви екранів
2. **H2 (Заголовок другого рівня)** — 20-24px, Semibold — назви секцій
3. **H3 (Заголовок третього рівня)** — 16-18px, Medium — підзаголовки
4. **Body (Основний текст)** — 14-16px, Regular — основний контент
5. **Caption / Small** — 12px, Regular — допоміжний текст, підписи

**Сервіси для створення типографічної ієрархії:**
- **Type Scale** (typescale.com) — генерація шкали розмірів шрифтів
- **Modular Scale** — розрахунок пропорційних співвідношень
- **Google Fonts** — підбір шрифтових пар
- **Font Pair** — рекомендації поєднань шрифтів

## Відповіді на контрольні запитання

### 1. У чому різниця між шрифтами Serif та Sans Serif? Який з них краще використовувати для довгих текстів у мобільному застосунку?

**Різниця між Serif та Sans Serif:**

| Характеристика | Serif (Антиква) | Sans Serif (Гротеск) |
|----------------|-----------------|---------------------|
| **Зовнішній вигляд** | Мають декоративні зарубки на кінцях літер | Не мають зарубок |
| **Асоціації** | Традиція, надійність, преміальність, класика | Сучасність, технологічність, мінімалізм |
| **Читабельність** | Краще для друкованих текстів | Краще для екранів |
| **Приклади** | Times New Roman, Georgia, Playfair Display | Inter, Roboto, Helvetica, Open Sans |

**Рекомендація для мобільного застосунку:**

Для довгих текстів у мобільному застосунку **краще використовувати Sans Serif** з наступних причин:

1. **Краща читабельність на екранах:** Беззарубкові шрифти спеціально розроблені для відображення на цифрових пристроях
2. **Менший розмір пікселів:** На маленьких екранах мобільних пристроїв зарубки Serif можуть виглядати розмитими
3. **Швидше сканування:** Sans Serif шрифти легше сприймаються при швидкому читанні
4. **Сучасний вигляд:** Більшість сучасних мобільних застосунків використовують Sans Serif для консистентності
5. **Рекомендації Apple та Google:** iOS (San Francisco) та Android (Roboto) використовують Sans Serif як системні шрифти

**Винятки:**
- Для преміальних застосунків (журнали, книги) можна використовувати Serif для заголовків
- Для коротких текстів (лейбли, теги) обидва типи підходять

### 2. Який тип палітри варто обрати, якщо ви хочете створити дизайн, що «кричить» про знижки або акції?

**Рекомендована палітра: Complementary (Комплементарна)**

**Обґрунтування:**

1. **Максимальний контраст:** Комплементарні кольори (розташовані навпроти в колірному колі) створюють найсильніший візуальний контраст
2. **Привертання уваги:** Контрастні поєднання (червоний-зелений, синій-помаранчевий, жовтий-фіолетовий) одразу привертають погляд
3. **Енергія та динаміка:** Такі палітри створюють відчуття руху та терміновості
4. **Тригер дії:** Контрастні кольори стимулюють користувача до негайних дій

**Приклади для знижок та акцій:**

- **Червоний + Білий:** Класичне поєднання для SALE (червоний — терміновість, білий — чистота)
- **Помаранчевий + Синій:** Високий контраст, асоціюється з енергією та довірою
- **Жовтий + Чорний:** Максимальна видимість (використовується для попереджувальних знаків)

**Додаткові рекомендації:**
- Використовуйте яскравий акцентний колір для кнопок CTA (Call to Action)
- Додавайте анімацію (мерехтіння, пульсацію) для посилення ефекту
- Використовуйте великі розміри шрифтів для цін та відсотків знижок
- Додавайте елементи терміновості ("Тільки сьогодні!", "Обмежена кількість!")

### 3. Які додаткові маркери (крім кольору) ви використаєте для повідомлення про успішну дію або помилку?

**Маркери для успішної дії (Success State):**

1. **Іконка галочки (✓)** — універсальний символ успіху
2. **Текстове повідомлення** — чітке описання результату ("Дедлайн успішно додано!")
3. **Анімація** — легка анімація галочки або появи повідомлення
4. **Звуковий сигнал** — приємний звук підтвердження (опціонально)
5. **Haptic feedback** — вібрація на мобільних пристроях
6. **Зміна стану елементу** — кнопка переходить у стан "Готово"
7. **Перехід на наступний екран** — автоматичне перенаправлення

**Маркери для помилки (Error State):**

1. **Іконка хрестика (✕) або знак оклику (!)** — універсальні символи попередження
2. **Текстове повідомлення** — конкретний опис помилки ("Введіть коректну дату")
3. **Червона рамка поля** — виділення поля з помилкою
4. **Анімація тряски** — легка тряска поля для привертання уваги
5. **Звуковий сигнал** — попереджувальний звук (опціонально)
6. **Haptic feedback** — вібрація помилки
7. **Підказка (Tooltip)** — додаткова інформація про виправлення
8. **Контрастний фон** — світло-червоний фон для повідомлення

**Приклад реалізації:**

```
Успіх:
[✓] Дедлайн "Курсова робота" додано!
Зелена іконка + зелений текст + анімація галочки

Помилка:
[!] Дата не може бути в минулому
Червона іконка + червоний текст + тряска поля
```
