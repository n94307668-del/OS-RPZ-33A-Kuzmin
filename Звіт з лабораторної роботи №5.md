## Відповіді на контрольні запитання

### 1. Як можна переглянути шлях до домашньої директорії користувача за допомогою команди `echo`?

Існує два способи переглянути шлях до домашньої директорії:

**Спосіб 1: використання змінної оточення `$HOME`**

```bash
echo $HOME
```

**Спосіб 2: використання символу `~` (тильда)**

```bash
echo ~
```

### 2. Чи можна переглянути вміст кореневого каталогу, перебуваючи у домашньому каталозі без переходу?

Так, це можливо. Для цього потрібно вказати шлях до каталогу як аргумент команди `ls`:

```bash
ls /
```

Ця команда відобразить вміст кореневого каталогу, не змінюючи поточну директорію. Аналогічно можна використовувати команду `ls -l /` для детальнішого перегляду.

### 3. Яким чином в терміналі можна додати інформацію в порожній файл?

Існує декілька способів додати інформацію в порожній файл:

**Спосіб 1: використання команди `echo` з перенаправленням виводу (`>` )**

```bash
echo “Текст для запису” > filename
```

**Спосіб 2: використання текстового редактора (nano, vim, gedit тощо)**

```bash
nano filename
```

**Спосіб 3: використання команди `cat` з перенаправленням**

```bash
cat > filename
```

### 4. Як скопіювати та видалити існуючий каталог? Чи буде відмінність в командах, якщо каталог буде не порожній?

**Копіювання каталогу:**

Для копіювання каталогу використовується команда `cp` з опцією `-R` (рекурсивне копіювання):

```bash
cp -R /dzherelo/katalog /priznachennya/
```

**Видалення каталогу:**

Для видалення каталогу використовуються дві команди:

1. `rm -r katalog` - видаляє каталог рекурсивно
2. `rmdir katalog` - видаляє лише порожній каталог

**Відмінність:**

Так, відмінність є. Команда `rmdir` працює лише з порожніми каталогами. Якщо каталог містить файли, `rmdir` видасть помилку. Команда `rm -r` видаляє каталог разом із усім вмістом.

### 5. У якому з наведених прикладів відбувається переміщення файлу? його перейменування? одночасно обидві дії?

- `mv /work/tech/comp.png /Desktop`
  - переміщення файлу з `/work/tech/` до `/Desktop` (зі зміною розташування)
- `mv /work/tech/comp.png /work/tech/my_car.png`
  - перейменування файлу (з `comp.png` на `my_car.png`) в тому ж каталозі
- `mv /work/tech/comp.png /Desktop/computer.png`
  - одночасно і переміщення (з `/work/tech/` до `/Desktop/`), і перейменування (з `comp.png` на `computer.png`)

### Лабораторна робота 7: Navigating the Filesystem

| Команда           | Опис                                        |
| :---------------- | :------------------------------------------ |
| `pwd`             | Виводить поточну робочу директорію (print working directory) |
| `echo $HOME`      | Виводить шлях до домашньої директорії користувача |
| `cd /`            | Перехід до кореневого каталогу (/)          |
| `cd`              | Перехід до домашньої директорії             |
| `cd /home`        | Перехід до каталогу /home                   |
| `cd ~`            | Перехід до домашньої директорії (символ ~)  |
| `cd ~root`        | Перехід до домашньої директорії користувача root |
| `cd /usr/bin`     | Перехід до каталогу /usr/bin                |
| `cd /usr/share/doc` | Перехід до каталогу документації            |
| `cd bash`         | Перехід до підкаталогу bash                 |
| `cd ..`           | Перехід до батьківського каталогу           |
| `cd ../dict`      | Перехід до каталогу dict у батьківському каталозі |
| `ls`              | Вивести вміст поточного каталогу            |
| `ls -a`           | Вивести вміст каталогу разом із прихованими файлами |
| `ls -l /etc/hosts` | Детальний перегляд файлу /etc/hosts         |
| `ls -R /etc/udev` | Рекурсивний перегляд /etc/udev              |
| `ls -d /etc/s*`   | Вивести каталоги /etc, що починаються з s   |
| `ls -d /etc/????` | Вивести каталоги /etc з 4-ма символами      |
| `ls -d /etc/[abcd]*` | Вивести каталоги /etc, що починаються з a, b, c або d |

### Лабораторна робота 8: Managing Files and Directories

| Команда           | Опис                                        |
| :---------------- | :------------------------------------------ |
| `echo *`          | Вивести усі файли та каталоги поточного каталогу |
| `echo D*`         | Вивести все, що починається з D             |
| `echo *s`         | Вивести все, що закінчується на s           |
| `echo D*n*s`      | Вивести шаблон D*n*s                        |
| `echo ??????`     | Вивести все з 6 символів                    |
| `echo D????????`  | Вивести все, що починається з D і має 9 символів |
| `echo ?????*s`    | Вивести шаблон: 5 символів + будь-що + s    |
| `echo [DP]*`      | Вивести все, що починається з D або P       |
| `echo [!DP]*`     | Вивести все, крім того, що починається з D чи P |
| `echo [D-P]*`     | Вивести все, що починається з літер від D до P |
| `echo [!D-P]*`    | Вивести все, крім літер від D до P          |
| `cp /etc/hosts hosts` | Копіювати /etc/hosts у файл hosts           |
| `cp -v /etc/hosts hosts` | Копіювати з виводом (verbose)               |
| `cp -p /etc/hosts ~` | Копіювати з збереженням атрибутів           |
| `cp -R /etc/udev Myetc` | Рекурсивне копіювання каталогу              |
| `rm hosts`        | Видалити файл hosts                         |
| `rm -r Myetc`     | Видалити каталог Myetc рекурсивно           |
| `mkdir Myetc`     | Створити каталог Myetc                      |
| `touch premove`   | Створити порожній файл premove              |
| `mv premove postmove` | Перейменувати файл premove у postmove       |
